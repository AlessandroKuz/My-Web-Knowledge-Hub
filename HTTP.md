# HTTP
An #HTTP request typically returns various type of files:
- [[HTML]]
- [[CSS]]
- [[JavaScript]]
- Extra Media files, like images, videos, audios, PDFs, and so on
The assembling of the pieces into a web page is called #Rendering.
